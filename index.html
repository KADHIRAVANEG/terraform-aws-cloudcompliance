Features

Everything in one CLI

Five commands cover the full SOC2 compliance lifecycle — from provisioning to audit evidence.

📋
cloudcompliance report
Reads terraform.tfstate, maps resources to SOC2 controls, generates JSON + Markdown evidence report. Saves score to history automatically.
v1.0.0
🔍
cloudcompliance drift
Compares live AWS state against Terraform state. Detects deleted or unmanaged resources. Fires SNS alert automatically.
v1.2.0
📈
cloudcompliance history
Every report run saved to SQLite. View compliance score trend over time — exactly what SOC2 Type II auditors need as evidence.
v1.3.0
🤖
cloudcompliance ask
AI assistant powered by NVIDIA NIM. Reads your actual tfstate and compliance reports. Answers questions about YOUR real infrastructure.
v1.3.0
🔧
cloudcompliance remediate
Auto-fixes low-risk drift instantly. Opens GitHub PR with exact remediation steps for high-risk changes. Full audit trail in remediation_log.json.
v1.4.0 NEW
🛡️
CI/CD Compliance Gate
GitHub Actions blocks every non-compliant PR. Runs Terraform validate, Checkov security scan, and full SOC2 compliance check automatically.
Always on
💬
Slack notifications
Drift alerts, compliance scores and PR notifications sent to your Slack channel automatically. Free webhook, no paid plan needed.
v1.6.0 NEW
⏰
cloudcompliance schedule
Runs drift detection automatically every hour. Sends Slack alert when drift is found. Runs until Ctrl+C.
v1.6.0 NEW

Quick start

Deploy in 3 commands

Requirements: Terraform, Docker, Python 3.9+

quick start
# Install CLI
$ pip install cloudcompliance

# Start LocalStack (free local AWS)
$ docker run --rm -d -p 4566:4566 localstack/localstack:3.4.0

# Deploy all 10 SOC2 controls
$ make deploy
Apply complete! Resources: 46 added, 0 changed, 0 destroyed.

# Generate compliance evidence
$ make report
SOC2 Compliance Score: 100% (10/10 controls passing)
Score saved to history: 100% (10/10 controls)

# Setup AI assistant
$ export NVIDIA_API_KEY="your-key"
$ cloudcompliance ask "am I ready for a SOC2 audit?"
